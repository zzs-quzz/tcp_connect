# 记录日志
## 1. 项目介绍
云深处绝影X30 Pro机器狗为例，在Linux系统中，使用c++实现tcp连接。
## 2. 使用方法
本项目开发平台信息如下：
- 系统版本：Ubuntu 20.04
- ROS_DISTRO：Noetic
### 2.1 下载
使用git clone下载安装
```shell
git clone https://github.com/zzs-quzz/tcp_connect.git
```
### 2.2 安装依赖
无需依赖安装
### 2.3 运行
1. 构建功能包
```
catkin_make
```
2. 运行测试程序
```
./devel/lib/tcp_connect/tcp_connect
```
